📱 iPhoneでの起動方法

  • 方法①：GitHub Pages（推奨・無料）
  • 1. PhysioCapture_Pro.html を GitHub リポジトリにアップロード
  • 2. Settings → Pages → Source を「main branch」に設定
  • 3. 発行されたURLをiPhoneのSafariで開く
  • 方法②：Macからローカル配信
  • 1. 「PhysioCapture_サーバー起動.command」をダブルクリック
  • 2. iPhoneをMacと同じWi-Fiに接続
  • 3. 表示されたURLをSafariで開く（証明書警告は「訪問」を選択）

📱 基本的な使い方

  • 1. 「カメラ開始」ボタンをタップ
  • 2. 正面または側面の撮影モードを選択
  • 3. 患者さんを画面中央に配置（全身が入るように）
  • 4. 骨格が自動検出され関節角度が表示されます
  • 5. 「録画開始」で動画に骨格を重ねて保存できます
  • 6. 「グラフ」タブで時系列グラフとCSVエクスポート

📐 測定される関節角度

  • 正面（前額面）
  • 左右 膝関節角度（外反・内反評価）
  • 左右 股関節外転角
  • 左右 肩関節外転角
  • 体幹側屈角
  • 側面（矢状面）
  • 左右 膝関節屈曲角
  • 左右 股関節屈曲角
  • 左右 肘関節屈曲角
  • 左右 肩関節屈曲角

⚠️ 注意事項

  • 明るい場所での撮影を推奨
  • 患者さんの全身が画面に入るようにしてください
  • HTTPS環境またはlocalhost上で動作します
  • 本アプリは理学療法士の補助ツールです。診断目的に使用しないでください

🤖 技術情報

Google MoveNet（TensorFlow.js）を使用した姿勢推定。17個のランドマーク（関節点）から関節角度を計算。OpenPoseと同等のCOCO姿勢推定アーキテクチャを採用。
